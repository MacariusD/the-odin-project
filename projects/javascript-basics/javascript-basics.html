Javascript Basics Projects from The Odin Project

FizzBuzz

This project displays numbers from 1 to a user selected number.

Numbers dividable by 3 will display Fizz
Numbers dividable by 5 will display Buzz
Numbers dividable by both will display FizzBuzz

Rock, Paper, Scissors

This project has the user play "Rock, Paper, Scissors" vs the computer.

Press the “Begin Game” button to start.
Press the “Rock”, “Paper”, or “Scissors” button to enter your choice.
The computer will randomly select an option to play against you.
The winner will be displayed.
This is a best of 5 rounds game.
